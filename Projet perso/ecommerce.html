Mini boutique en ligne : catalogue, page produit et panier

Projet e-commerce réalisé pour mettre en pratique la création d’un catalogue de produits, l’affichage d’une page détail et la gestion d’un panier (ajout, quantité, total). L’objectif était d’avoir une interface propre, responsive et une logique simple.

Fonctionnalités

  • Affichage d’un catalogue de produits (cartes + infos essentielles)
  • Page détails produit (prix, description, visuel)
  • Panier : ajout/suppression, gestion des quantités, total automatique
  • Mise en page responsive (mobile / desktop)

Ce que j’ai appris

  • Structurer un projet (pages, composants, styles)
  • Gérer une logique de panier (quantités, total, affichage)
  • Améliorer l’UX : boutons clairs, layout propre
  • Rendre un rendu cohérent

Technos utilisées

  • HTML / CSS : interface, composants, responsive
  • PHP (si applicable) : pages dynamiques + logique côté serveur
  • Sessions (si applicable) : panier persistant

À améliorer (suite)

  • Recherche / filtres / tri
  • Base de données (produits, commandes)
  • Espace compte + historique
